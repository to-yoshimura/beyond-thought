# NEURAL VOID

An interactive generative artwork inspired by neural networks, memory, and the layered structure of consciousness.

## Experience

- Generate between 50 and 2,000 neural nodes
- Stimulate nodes and watch signals propagate through the network
- Reinforce frequently used pathways through neural plasticity
- Form temporary thought crystals through repeated stimulation
- Move through Surface, Memory, and Subconscious layers
- Explore a procedural spatial soundscape generated with the Web Audio API

## Local preview

Open `index.html` directly, or run a local static server:

```sh
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.
